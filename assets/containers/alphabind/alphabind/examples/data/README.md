# Tutorial Datasets and Default Output Directories

## Directory quick-reference
**NOTE**: Items in `[brackets]` are default paths of output artifacts created by our examples and are not versioned in this repo.
```
├── embeddings:                                 Default output directory for embeddings generated by `tutorial_1_finetuning_alphabind.ipynb`
│   └── raw:                                    Stores individual `.npy` embeddings generated per-sequence by `tutorial_1_finetuning_alphabind.ipynb`
├── models:                                     Default output directory for models generated by `tutorial_1_finetuning_alphabind.ipynb`
│   └── logs:                                   Default output directory for PyTorch Lightning logs generated by model training using `tutorial_1_finetuning_alphabind.ipynb`
├── preprocessed
│   └── [train_data.csv]:                       Generated by `tutorial_1_finetuning_alphabind.ipynb`
├── raw
│   └── DSM_enrichment_full_seqs.csv:           Original dataset from _Mason et al._ (see `tutorial_1_finetuning_alphabind.ipynb` for full citation)
└── results:                                    Default output directory for `tutorial_2_optimization_alphabind.ipynb`
    ├── [all_unique_candidates.csv]:            Default output path for the unique, merged sequences from all generations
    ├── [last_generation_optimized_seqs.csv]:   Default output path for sequences from the last generation
    └── [optimization_steps]:                   Default output directory for storing optimized sequences from intermediate generations
```
